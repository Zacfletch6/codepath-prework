import numpy as np
import IPython.display as display
from matplotlib import pyplot as plt
import io
import base64

ys = 200 + np.random.randn(100)
x = [x for x in range(len(ys))]

fig = plt.figure(figsize=(4, 3), facecolor='w')
plt.plot(x, ys, '-')
plt.fill_between(x, ys, 195, where=(ys > 195), facecolor='y', alpha=0.45)
plt.title("Cost Over Time", fontsize=10)
plt.xlabel( "time" )
plt.ylabel("cost")

data = io.BytesIO()
plt.savefig(data)
image = F"data:image/png;base64,{base64.b64encode(data.getvalue()).decode()}"
alt = "Sample Visualization"
#display.display(display.Markdown(F"""![{alt}]({image})"""))
plt.show()
plt.close(fig)

Task III

Changes to the graph and why they're important:

  • I changed the title Sample Visualization to Cost Over Time by changing plt.title("Sample Visualization", fontsize=10) to plt.title("Cost Over Time", fontsize=10). I added axis labels Cost and Time by using the plt.xlabel and plt.ylabel methods. Labels are important so decision makers can easily understand the meaning of the graph at a glance.

  • In the code for the graph above, facecolor = 'g' has been changed to facecolor = 'y', changing the color of the graph from green to yellow. I changed the alpha argument in the parameters for plt.fill_between from .6 to .45. After some experimentation, I believe the alpha controls the opacity of the fill color.

  • I commented out the display.display expression and added the plt.show() instead. The .show method is easier to write and easier for other programmers to read.
